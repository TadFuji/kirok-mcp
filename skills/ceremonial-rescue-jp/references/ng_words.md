### 重ね言葉（弔事と共通）

弔事と同じく、重ね重ね / たびたび / またまた / いよいよ / ますます もNG。

### 句読点ルール

結婚式の招待状、メッセージカードには **句読点（、。）を使わない**。
理由: 「句読点 = 区切り・終わり」を連想するため。
改行や空白で代替する。

## 宗派別NG

### 浄土真宗

| NGワード | 代替表現 | 理由 |
|----------|----------|------|
| ご冥福 | お悔やみ申し上げます / 哀悼の意を表します | 浄土真宗では「冥途（暗い世界）」の概念がない。即身成仏の教義 |
| 冥福 | お悔やみ / 哀悼の意 | 同上 |
| 御霊前 | 御仏前 | 浄土真宗は即成仏。「霊」の状態を経ない |
| 草葉の陰 | お浄土から | 草葉の陰は幽霊のイメージ。浄土真宗は浄土へ往生 |
| 天国 | お浄土 / 極楽浄土 | 天国はキリスト教の概念 |
| 安らかにお眠りください | 安らかにお旅立ちください | 浄土真宗では故人は眠っていない（浄土で活動中） |

> **⚠️ 「ご冥福をお祈りします」は日本の弔事で最もよく使われるフレーズだが、浄土真宗では禁忌。これが最も多い間違い。**

### 神式

| NGワード | 代替表現 | 理由 |
|----------|----------|------|
| 御仏前 | 御玉串料 / 御榊料 | 「仏」は仏教用語。神道には使わない |
| 成仏 | （使わない — 仏教用語） | 同上 |
| 供養 | 追悼 / お祀り | 「供養」は仏教用語 |

### キリスト教

| NGワード | 代替表現 | 理由 |
|----------|----------|------|
| 御霊前 | 御花料 | キリスト教式の表書き |
| 御仏前 | 御花料 | 仏教用語は不適切 |

## 使い方ガイド

### エージェントとしての使い方

1. **メッセージ生成前**: このファイルのNGワードリストを参照し、使用を避ける
2. **メッセージ生成後**: `scripts/ng_word_checker.py` でプログラム的に最終チェック
3. **NG検出時**: このファイルの「代替表現」列から適切な言い換えを選択
4. **宗派判明時**: 宗派別NGセクションを追加でチェック
5. **宗派不明時**: 全宗派で安全な表現を選ぶ（≒浄土真宗のルールに従うのが最も安全）

### `ng_word_checker.py` との連携

```bash
# 弔事メッセージのチェック
python ng_word_checker.py --text "メッセージ" --type funeral

# 宗派指定チェック
python ng_word_checker.py --text "メッセージ" --type funeral --sect 浄土真宗

# 慶事メッセージのチェック
python ng_word_checker.py --text "メッセージ" --type wedding

# お祝いメッセージのチェック
python ng_word_checker.py --text "メッセージ" --type celebration

# 全タイプ横断チェック
python ng_word_checker.py --text "メッセージ" --type all
```
