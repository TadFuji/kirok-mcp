## 3層判断フレーム

会社関係の冠婚葬祭では、対応レベルを3層で判断する。

| 層 | 条件 | 対応内容 |
|----|------|----------|
| **個人レベル** | 個人的な付き合いがある | 個人名で香典/ご祝儀 + メッセージ |
| **部署レベル** | 同じ部署・チーム | 部署名義で合同の香典/ご祝儀 + 弔電/祝電 + 供花 |
| **会社レベル** | 役員 / 重要取引先 / 社内規定による | 会社名義で弔電/祝電 + 供花 + 必要に応じ代表参列 |

### 判断フロー

```
故人/当事者との関係は？
├── 直接の同僚/上司/部下 → 個人 + 部署レベル
├── 同僚/上司/部下の家族 → 個人 + 部署レベル（規模に応じ）
├── 他部署の社員 → 部署レベル（社内通知のみの場合も）
├── 取引先の担当者本人 → 部署レベル + 会社レベル検討
├── 取引先の役員/社長 → 会社レベル
└── 社員の遠い親族 → 個人レベル（任意）
```

## 社内通知テンプレート

### 弔事 — メール版

```
件名: 【訃報】○○部 △△さん ご尊父様ご逝去のお知らせ

社員各位

○○部 △△さんのお父様が○月○日にご逝去されました。
謹んでお悔やみ申し上げます。

■ 通夜:  ○月○日（○）○時〜 △△斎場
■ 告別式: ○月○日（○）○時〜 同上
■ 宗派:  ○○宗（確認中の場合は「確認中」）

■ 香典について
  □ 部署として合同で対応（一人○○円を△△まで）
  □ 個人判断
  □ ご辞退（香典辞退の意向あり）

■ 供花・弔電の手配
  □ 部署名義で手配済み
  □ 会社名義で手配済み(総務部)

お問い合わせ: ○○部 △△（内線○○）
```

### 弔事 — チャット版（Slack/Teams用）

```
【訃報のお知らせ】
○○部 △△さんのお父様がご逝去されました。

通夜: ○月○日 ○時〜 △△斎場
告別式: ○月○日 ○時〜

香典: 部署合同で対応します（詳細は別途連絡）
※この件に関する△△さんへの直接連絡はお控えください
```

### 慶事 — メール版

```
件名: ○○部 △△さん ご結婚のお知らせ

社員各位

○○部 △△さんが○月○日にご結婚されました。
心よりお祝い申し上げます。

■ ご祝儀について
  □ 部署として合同で対応（詳細は△△まで）
  □ 個人判断

お問い合わせ: ○○部 △△
```

## HR確認チェックリスト

弔事が発生した際にHR/総務に確認すべき事項:

- [ ] 会社の慶弔規定（慶弔見舞金の金額・条件）
- [ ] 忌引休暇の日数（配偶者/親/祖父母/兄弟 で異なる）
- [ ] 弔電・供花の社内手配手順
- [ ] 社長名義の弔電が必要か（役職・関係性による）
- [ ] 代表参列者の選定（上長 or 総務）
- [ ] 社内通知の範囲（全社/部署/チームのみ）
- [ ] 香典の合同集金を誰が取りまとめるか

### 忌引休暇の一般的な日数（法定外、会社判断）

| 対象 | 一般的な日数 |
|------|-------------|
| 配偶者 | 10日 |
| 父母（実父母・義父母） | 7日 |
| 子 | 5日 |
| 祖父母 | 3日 |
| 兄弟姉妹 | 3日 |
| 叔父叔母 | 1日 |
| 配偶者の祖父母 | 1〜3日 |

> ※ 法定の規定はないため、必ず自社の就業規則を確認する

## 供花弔電の手配

### 供花の手配フロー

```
1. 斎場名・住所を確認
2. 斎場の指定花屋があるか確認（多くの斎場は指定あり）
   ├── 指定あり → その花屋に電話注文
   └── 指定なし → 近隣の花屋 or オンライン花サービス
3. 名札の表記を決める
   - 個人: 「△△ △△」
   - 部署: 「○○株式会社 ○○部一同」
   - 会社: 「○○株式会社 代表取締役 △△ △△」
4. 予算: 一般的に 15,000〜30,000円
5. 到着日時: 通夜開始の2〜3時間前まで
```

### 弔電の手配

- **NTT（115番）** or 各社の電報サービス
- 文例は電報サービスに用意あり（関係性を伝えて選択）
- 名義: 会社名 or 部署名 + 代表者名
- 到着: 通夜の前日〜当日午前中

## 慶事の会社対応

### 結婚祝い

- **部署名義**: 部署メンバーから均等に集金（一人2,000〜5,000円程度）、合同で贈答品 or 現金
- **会社名義**: 慶弔規定に基づく祝金 + 祝電
- **個人**: 出席する場合はご祝儀を個人で用意

### 出産祝い

- 部署名義で合同の出産祝いが一般的
- 金額: 一人1,000〜3,000円程度の集金
- プレゼント選び: 本人の希望を聞く（ベビー用品の重複を避ける）

### 注意事項

- 妊娠・出産の話題は本人が公表してから
- 育休中の社員への通知タイミングは配慮する
- 性別の公表にも注意（本人が望まない場合あり）
