## のし袋の書き方

### 弔事 — 表書き（宗派別）

| 宗派 | 表書き | 水引 |
|------|--------|------|
| 仏式（一般） | 御霊前（四十九日前）/ 御仏前（四十九日後） | 黒白・双銀の結び切り |
| 浄土真宗 | 御仏前（※御霊前NG — 即身成仏の教義） | 黒白・双銀の結び切り |
| 神式 | 御玉串料 / 御榊料 | 黒白の結び切り |
| キリスト教（カトリック） | 御花料 / 御ミサ料 | 白無地封筒（水引なし）or 十字架付き |
| キリスト教（プロテスタント） | 御花料 / 忌慰料 | 白無地封筒（水引なし）or 白百合柄 |
| 無宗教 / 宗派不明 | 御霊前（最も無難） | 黒白の結び切り |

### 弔事 — のし袋の書き方ビジュアル

```
┌──────────────────┐
│                  │
│    御 霊 前      │  ← 表書き（薄墨で書く）
│                  │
│    山田 太郎     │  ← フルネーム（薄墨で書く）
│                  │
│ ━━━━━━━━━━━━━━ │  ← 水引（黒白 結び切り）
│                  │
└──────────────────┘

【中袋（内袋）】
┌──────────────────┐  ┌──────────────────┐
│ （表面）         │  │ （裏面）         │
│                  │  │                  │
│  金 伍仟圓 也   │  │  〒xxx-xxxx      │
│                  │  │  東京都○○区...  │
│                  │  │  山田 太郎       │
└──────────────────┘  └──────────────────┘
```

- **薄墨**: 弔事の表書きは薄墨（悲しみの涙で墨が薄まったことを表す）
- **金額**: 漢数字（壱/弐/参/伍/拾/萬/圓）で記入
- **向き**: お札の裏面が前に来るように入れる（顔を伏せる）

### 慶事 — 表書き

| 場面 | 表書き | 水引 |
|------|--------|------|
| 結婚式 | 寿 / 御祝 / 御結婚御祝 | 金銀 or 紅白の結び切り（10本） |
| 出産祝い | 御祝 / 御出産御祝 | 紅白の蝶結び |
| 入学祝い | 御祝 / 御入学御祝 | 紅白の蝶結び |
| 七五三 | 御祝 | 紅白の蝶結び |
| 還暦祝い | 寿 / 御祝 | 紅白の蝶結び or 結び切り |
| お中元 | 御中元 | 紅白の蝶結び |
| お歳暮 | 御歳暮 | 紅白の蝶結び |
| 快気祝い | 快気祝 / 快気内祝 | 紅白の結び切り（繰り返さない意味） |

### 慶事 — 祝儀袋ビジュアル

```
┌──────────────────┐
│      ◇◇◇       │  ← のし飾り
│                  │
│     寿          │  ← 表書き（濃い墨で書く）
│                  │
│    山田 太郎     │  ← フルネーム
│                  │
│ ═══════════════ │  ← 水引（金銀 結び切り 10本）
│                  │
└──────────────────┘
```

- **濃い墨**: 慶事は濃い墨（喜びを表す）
- **新札**: 必ず新札を用意（事前に準備していた＝楽しみにしていた意味）
- **向き**: お札の表面（顔）が前に来るように入れる

### 水引の使い分け

| 水引の種類 | 意味 | 使う場面 |
|-----------|------|---------|
| **結び切り** | 二度と繰り返さない | 結婚式、弔事、快気祝い、お見舞い |
| **蝶結び（花結び）** | 何度あっても嬉しい | 出産、入学、お中元、お歳暮、七五三 |

## 宗派別参拝手順

### 焼香（仏式）

| 宗派 | 焼香回数 | 補足 |
|------|---------|------|
| 真言宗 | 3回 | 額に押し戴く |
| 天台宗 | 1〜3回 | 特に定めなし |
| 浄土宗 | 1〜3回 | 額に押し戴く |
| 浄土真宗（本願寺派） | 1回 | 額に押し戴かない |
| 浄土真宗（大谷派） | 2回 | 額に押し戴かない |
| 曹洞宗 | 2回 | 1回目は額に押し戴く、2回目は戴かない |
| 日蓮宗 | 1〜3回 | 額に押し戴く |
| 臨済宗 | 1回 | 額に押し戴く |
| **宗派不明** | **1回** | **額に軽く押し戴く（最も無難）** |

#### 焼香の基本手順

1. 遺族に一礼
2. 焼香台に進み、合掌・一礼
3. 右手の親指・人差し指・中指で抹香をつまむ
4. （宗派による）額の高さに押し戴く
5. 香炉に静かに落とす
6. 宗派の回数を繰り返す
7. 合掌・一礼
8. 遺族に一礼して戻る

### 玉串奉奠（神式）

1. 神職から玉串（榊）を受け取る（右手で根元、左手で葉先を支える）
2. 胸の高さに持ち、祭壇に進む
3. 玉串を時計回りに90度回す（根元を手前に）
4. さらに時計回りに90度（根元を祭壇側に向ける）
5. 玉串案（台）に置く
6. 二礼二拍手一礼（※弔事の拍手は「しのび手」＝音を立てない）

### 献花（キリスト教）

1. 花を受け取る（右手で茎、左手で花を支える）
2. 献花台に進み一礼
3. 花を時計回りに回し、茎を祭壇側に向ける
4. 献花台に置く
5. 黙祷（合掌ではない）
6. 一礼して戻る

## 口頭セリフ集

### 弔事 — 場面別

#### 受付で

| 距離感 | セリフ |
|--------|--------|
| 普通 | 「このたびはご愁傷さまでございます」 |
| 形式的 | 「このたびは誠にご愁傷さまでございます。心よりお悔やみ申し上げます」 |
| 会社関係 | 「このたびは突然のことで…心よりお悔やみ申し上げます」 |

#### 遺族への声かけ

| 距離感 | セリフ |
|--------|--------|
| 親しい | 「突然のことで言葉もありません。何かできることがあればおっしゃってください」 |
| 普通 | 「このたびはご愁傷さまでございます。お力落としのないように」 |
| 形式的 | 「心よりお悔やみ申し上げます。安らかなご冥福をお祈りいたします」 |

> **⚠️ 浄土真宗の場合**:「ご冥福」は使えない →「心よりお悔やみ申し上げます」のみ

#### 通夜振る舞いで

- 短時間で退席する:「お招きいただきありがとうございます。お言葉に甘えていただきます」
- 退席時:「このあたりで失礼いたします。どうぞご無理なさらないように」

#### 電話での弔意

「○○と申します。このたびはご愁傷さまでございます。突然のことで大変驚いております。お忙しいところ恐れ入りますが、お通夜のご予定を教えていただけますでしょうか」

#### 後日弔問

「ご連絡が遅くなり申し訳ございません。遅ればせながら、心よりお悔やみ申し上げます。お線香をあげさせていただいてもよろしいでしょうか」

### 慶事 — 場面別

#### 受付で

「本日はおめでとうございます」（一言で十分）

#### 新郎新婦への一言

| 距離感 | セリフ |
|--------|--------|
| 親しい | 「結婚おめでとう！本当に嬉しいよ。末永くお幸せに」 |
| 普通 | 「ご結婚おめでとうございます。お二人の幸せをお祈りしています」 |
| 形式的 | 「ご結婚おめでとうございます。末永いご多幸をお祈り申し上げます」 |

#### ご両親への一言

「本日はおめでとうございます。素敵なお式ですね」

## 招待状返信

### 出席する場合

```
【表面】
「行」を二重線で消して「様」に書き換え

【裏面】
「御出席」の「御」を二重線で消す
「御欠席」は全体を二重線で消す
「御芳名」の「御芳」を、「御住所」の「御」を二重線で消す
空きスペースに一言添える:
  「ご結婚おめでとうございます。喜んで出席させていただきます」
```

### 欠席する場合

```
「御欠席」の「御」を二重線で消す
「御出席」は全体を二重線で消す
必ず理由と祝辞を書く:
  「ご結婚おめでとうございます。やむを得ない事情により
   出席がかなわず、大変残念に思います。
   お二人の末永いお幸せをお祈り申し上げます」
```

- **不祝儀が理由**: 具体的に書かない（「やむを得ない事情」で濁す）
- **返信期限**: 招待状到着から1週間以内が理想
- **アレルギー**: 「お手数ですが、○○アレルギーがございますのでご配慮いただけますと幸いです」

## 服装持ち物

### 弔事

| アイテム | 基本ルール |
|----------|-----------|
| 服装（男性） | ブラックスーツ、白ワイシャツ、黒ネクタイ、黒靴下、黒靴 |
| 服装（女性） | 黒のワンピース or アンサンブル、黒ストッキング、黒パンプス |
| アクセサリー | パールのネックレス（一連のみ）。二連NG（不幸が重なる） |
| バッグ | 黒の布製、光沢なし |
| 持ち物 | 数珠（宗派により形が異なるが略式数珠で可）、ふくさ（紫 or 紺） |
| ハンカチ | 白 or 黒の無地 |
| NG | 殺生を連想させる素材（毛皮、爬虫類革）、光る素材、派手な時計 |

### 慶事

| アイテム | 基本ルール |
|----------|-----------|
| 服装（男性） | ダークスーツ or ブラックスーツ、白 or パステルカラーのシャツ |
| 服装（女性） | ドレス or ワンピース。白NG（花嫁の色）。全身黒もNG（喪服連想） |
| 靴 | つま先が隠れるパンプス。サンダル・ミュール・ブーツNG |
| バッグ | 小ぶりのパーティーバッグ |
| NG | 白い服（花嫁と被る）、黒タイツ、ファー（殺生）、大きなバッグ |

## 行動NG

### 弔事の行動NG

| NG行動 | 理由 |
|--------|------|
| 新札を包む | 不幸を予期していた印象 |
| 香典を直接手渡しする | 必ずふくさに包んで受付で渡す |
| 遺族に死因を詳しく聞く | プライバシー侵害。「突然のことで」で十分 |
| 故人の写真を撮る | 不謹慎 |
| 家族葬に不招待で参列する | 遺族の意向を最優先 |
| 通夜振る舞いを断る | 故人への供養として一口は箸をつける |
| 「頑張って」と言う | 遺族にプレッシャー。「ご無理なさらず」 |
| 大声で話す | 厳粛な場にふさわしくない |
| 香典を偶数・4/9の金額にする | 縁が切れる/死/苦を連想 |

### 慶事の行動NG

| NG行動 | 理由 |
|--------|------|
| 旧札を包む | 準備不足の印象 |
| スピーチで暴露話をする | 新郎新婦の面目を傷つける |
| 白い服で出席する（女性） | 花嫁の色と被る |
| 祝儀袋に句読点を使う | 「、」「。」は区切り・終わりを連想 |
| 招待状返信を遅らせる | 席次表の準備に影響 |
| 過度な写真撮影 | 式の進行を妨げる |
| スピーチを長くする（5分超） | ゲストの集中力低下 |
| 遅刻 | 開始30分前到着が理想 |

## スピーチ構成

### 基本構成（3〜5分、800〜1200字）

```
1. 挨拶・自己紹介（30秒）
   「ただいまご紹介にあずかりました、○○と申します。
    新郎の△△さんとは大学時代のサークル仲間です」

2. エピソード（2〜3分）
   - 新郎新婦との具体的なエピソードを1〜2つ
   - 人柄が伝わる温かいエピソード
   - NG: 暴露話、下ネタ、元カノ/元彼の話

3. 締め — はなむけの言葉（30秒）
   「お二人の末永い幸せを心よりお祈り申し上げます。
    本日は誠におめでとうございます」
```

### 時間×文字数の目安

| 時間 | 文字数 | 評価 |
|------|--------|------|
| 2分 | 500〜600字 | やや短い |
| 3分 | 800〜900字 | ★ちょうどよい |
| 5分 | 1200〜1500字 | 上限 |
| 7分以上 | 2000字超 | ❌長すぎる |

## 地域差

### 基本パターン（不変の文化的特徴）

| 地域 | 特徴 |
|------|------|
| **北海道** | 会費制の結婚式が一般的（ご祝儀不要の場合あり） |
| **関東** | 全国標準に最も近い |
| **関西** | 黄白の水引を使う地域あり。引き出物は複数品 |
| **東海（名古屋）** | 婚礼が派手（「名古屋婚」）。菓子まきの文化 |
| **九州** | 即日返し（当日に香典返し）が一般的な地域あり |
| **沖縄** | 相場が低め。結婚式は大規模（200〜300名）。清明祭（シーミー）の文化 |

### 適用ルール

1. ユーザーの地域が判明 → 地域差について注記する
2. 地域不明 → 関東（全国標準）をデフォルト
3. 地域差は「根本ルールの違い」ではなく「運用の差」として扱う
4. 最新の地域事情は Web 検索で補完する
