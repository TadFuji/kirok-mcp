# 喪中はがきテンプレート (Mourning Postcard)

**用途:** 喪中のため年賀状を出せない旨の通知（11月〜12月初旬に送付）
**出力形式:** .docx + .pdf
**必須変数:** `{{deceased_name}}`, `{{death_month}}`, `{{age}}`, `{{sender_name}}`, `{{sender_address}}`
**敬語レベル:** フォーマル

---

## テンプレート本文

```
喪中につき年末年始のご挨拶を
ご遠慮申し上げます

本年{{death_month}}月に {{relationship}} {{deceased_name}}が
{{age}}歳にて永眠いたしました

ここに本年中に賜りましたご厚情を深謝いたしますとともに
明年も変わらぬご交誼のほどお願い申し上げます

令和○年○月

〒{{postal_code}}
{{sender_address}}
{{sender_name}}
```

## 注意

- 句読点を使わない（慣例）
- 送付時期：11月中旬〜12月初旬（相手が年賀状を準備する前に届くように）
- 年齢は数え年が正式だが、満年齢でも近年は許容される
- 2親等以内（配偶者・父母・子・兄弟姉妹・祖父母・孫）が一般的な喪中の範囲
- 会社関係の年賀状は通常通り出してよい（個人的な関係のみ喪中扱い）
