# 後日お悔やみテンプレート (Belated Condolence)

**用途:** 訃報を後日知った場合のお悔やみの手紙・メール
**出力形式:** .docx
**必須変数:** `{{recipient_name}}`, `{{deceased_name}}`, `{{sender_name}}`
**敬語レベル:** フォーマル

---

## テンプレート：手紙（1ヶ月以内）

```
{{recipient_name}}様

突然のお便りをお許しください

{{deceased_name}}様のご逝去を遅ればせながら知り
驚きとともに深い悲しみを覚えております

ご生前には大変お世話になりながら
お見送りにも伺えず 誠に申し訳なく存じます

心ばかりのものを同封いたしましたので
ご霊前にお供えいただければ幸いに存じます

ご遺族の皆様のご健康を
心よりお祈り申し上げます

{{date}}
{{sender_name}}
```

## テンプレート：メール（1週間以内）

件名：○○様のご逝去について

```
{{recipient_name}}様

{{deceased_name}}様のご逝去を先ほど知り
急ぎご連絡いたしました

突然のことに言葉もございません
心よりお悔やみ申し上げます

もしよろしければ お線香をあげに伺いたいのですが
ご都合のよろしい日時をお教えいただけますでしょうか

何かお手伝いできることがございましたら
いつでもお声がけください

{{sender_name}}
```

## テンプレート：3ヶ月以上経過

```
{{recipient_name}}様

ご無沙汰しております

{{deceased_name}}様のことを今になって知り
遅ればせながらお悔やみ申し上げます

大変な時期にお力になれず申し訳ございませんでした
今後何かございましたらいつでもお声がけください

{{sender_name}}
```

## 注意

- 手紙には句読点を使わない（慣例）
- 香典を同封する場合は現金書留で送る
- 後日知った場合、自分を責める表現（「知らなかった自分が恥ずかしい」等）は遺族の負担になるので控える
