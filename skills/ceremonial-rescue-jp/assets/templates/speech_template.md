# スピーチ原稿テンプレート (Speech Template)

**用途:** 結婚式スピーチ原稿の骨格
**出力形式:** .docx
**必須変数:** `{{speaker_name}}`, `{{speaker_relationship}}`, `{{groom_name}}`, `{{bride_name}}`
**敬語レベル:** フォーマル（スピーチ体）

---

## 時間と文字数の目安

| 時間 | 文字数 |
|---|---|
| 3分 | 約900〜1,000字 |
| 5分 | 約1,500〜1,700字 |
| 7分 | 約2,100〜2,400字 |

---

## テンプレート：友人スピーチ（5分）

```
【自己紹介】（〜100字）
ただいまご紹介にあずかりました、
{{speaker_relationship}}の{{speaker_name}}と申します。

【お祝いの言葉】（〜100字）
{{groom_name}}さん、{{bride_name}}さん、
ご結婚おめでとうございます。
ご両家の皆様にも心よりお祝い申し上げます。

【エピソード①：出会い・人柄】（〜500字）
{{episode_1}}

【エピソード②：パートナーとの出会い後の変化】（〜500字）
{{episode_2}}

【はなむけの言葉】（〜200字）
{{closing_message}}

【結び】（〜100字）
おふたりの末永いお幸せを心よりお祈り申し上げまして、
お祝いの言葉とさせていただきます。
本日はまことにおめでとうございます。
```

## テンプレート：上司スピーチ（5分）

```
【自己紹介と関係】（〜100字）
ただいまご紹介にあずかりました、
{{groom_or_bride_name}}さんの上司にあたります{{speaker_name}}でございます。

【お祝いの言葉】（〜100字）
{{groom_name}}さん、{{bride_name}}さん、
ご結婚おめでとうございます。

【仕事でのエピソード】（〜600字）
{{work_episode}}

【人柄の紹介】（〜300字）
{{character_description}}

【パートナーへのメッセージ】（〜200字）
{{partner_message}}

【結び】（〜200字）
おふたりの新しい生活が実り多きものとなりますよう
心からお祈り申し上げまして、
私からのお祝いの言葉とさせていただきます。
本日はまことにおめでとうございます。
```

## スピーチのチェックリスト

- [ ] 忌み言葉が含まれていないか（ng_word_checker.py で確認）
- [ ] 時間内に収まるか（文字数チェック）
- [ ] 新郎新婦の名前を間違えていないか
- [ ] 暴露話・内輪ネタが含まれていないか
- [ ] 自分語りが長すぎないか
- [ ] 最初と最後にお祝いの言葉があるか
- [ ] 声に出して読んでみて自然か
