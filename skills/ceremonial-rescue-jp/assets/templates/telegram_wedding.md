# 祝電テンプレート (Wedding Telegram)

**用途:** 祝電文面の作成（欠席時にも使用）
**出力形式:** .docx
**必須変数:** `{{groom_name}}`, `{{bride_name}}`, `{{sender_name}}`
**敬語レベル:** フォーマル

---

## テンプレート：一般

```
ご結婚おめでとうございます
おふたりの門出を心よりお祝い申し上げます
笑顔あふれる温かいご家庭を
お築きになられますことを
お祈りいたします

{{sender_name}}
```

## テンプレート：友人向け（やや親しい）

```
{{groom_name}}さん {{bride_name}}さん
ご結婚おめでとうございます
おふたりの晴れの日を
心からお祝い申し上げます
いつまでも仲睦まじく
お幸せに

{{sender_name}}
```

## テンプレート：会社関係

```
ご結婚誠におめでとうございます
おふたりの人生の新たな門出を
心よりお祝い申し上げます
ご多幸とご健勝を
お祈りいたします

{{company_name}}
{{sender_title}} {{sender_name}}
```

## テンプレート：欠席のお詫び付き

```
ご結婚おめでとうございます
やむなく欠席となりましたが
おふたりの晴れの日を
心よりお祝い申し上げます
末永いお幸せをお祈りいたします

{{sender_name}}
```

## 注意

- 句読点を使わない
- 忌み言葉に注意（別れる、切る、去る、終わる 等）
- 重ね言葉NG（重ね重ね、たびたび、再び 等）
- 到着タイミング: 挙式の前日〜当日午前
- 宛先: 式場名 気付 新郎新婦名
