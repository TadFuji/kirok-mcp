# 弔電テンプレート (Funeral Telegram)

**用途:** 弔電文面の作成
**出力形式:** .docx
**必須変数:** `{{deceased_name}}`, `{{sender_name}}`
**敬語レベル:** 最フォーマル

---

## テンプレート：一般（仏式）

```
{{deceased_name}}様のご逝去の報に接し
心より哀悼の意を表します
ご遺族の皆様に
心からお悔やみを申し上げますとともに
故人のご冥福をお祈りいたします

{{sender_name}}
```

## テンプレート：浄土真宗対応

```
{{deceased_name}}様のご逝去を悼み
謹んでお悔やみ申し上げます
安らかにお旅立ちになられましたことを
心よりお祈り申し上げます

{{sender_name}}
```

## テンプレート：会社名義

```
{{deceased_name}}様のご逝去の報に接し
弊社社員一同 心より哀悼の意を表します
ご生前のご厚誼に深く感謝申し上げますとともに
故人の安らかなるご永眠をお祈り申し上げます

{{company_name}}
{{representative_title}} {{representative_name}}
```

## テンプレート：部署名義

```
{{deceased_name}}様のご逝去を悼み
{{department_name}}一同 謹んでお悔やみ申し上げます
ご冥福をお祈りいたします

{{company_name}} {{department_name}}一同
```

## 注意

- 弔電では句読点を使わない（文末に「。」を打たない）
- 浄土真宗では「ご冥福」「ご霊前」を使わない
- 宛先: 斎場名 気付 ○○家（喪主名がわかれば喪主宛）
- 到着タイミング: 通夜の前日〜当日午前
